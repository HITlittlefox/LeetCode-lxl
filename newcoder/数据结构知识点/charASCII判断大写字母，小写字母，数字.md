1. 判断一个字符串是不是大写字母
    ```java
    char c = 'S';
    System.out.println(c >= 'A' && c <= 'Z' ? true : false);
    ```
2. 判断一个字符串是不是小写字母
    ```java
    char c = 'S';
    System.out.println(c >= 'a' && c <= 'z' ? true : false);
    ```
3. 判断一个字符串是不是数字
    ```java
    char c = '2';
    System.out.println(c >= '0' && c <= '9' ? true : false);
    ```
